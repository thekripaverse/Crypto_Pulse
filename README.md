## Project Overview

CryptoPulse is a real-time cryptocurrency tracking dashboard built using Streamlit. The platform provides live market prices, market capitalization updates, interactive trend charts, and trading signals in a lightweight and intuitive interface. The project is designed to help users monitor crypto market movements and gain quick insights for analysis and learning purposes.

CryptoPulse integrates live data from the CoinGecko API and visualizes it using interactive charts to deliver a smooth and informative user experience.

---

## Key Features

* Live cryptocurrency price updates (auto-refreshing)
* Real-time market capitalization tracking
* Interactive 7-day price trend charts
* Trading signals and alerts for major cryptocurrencies
* Personalized dashboard layout
* Lightweight, fast, and responsive UI

## Installation and Setup

### 1. Clone the Repository

```bash
git clone https://github.com/Kripasree-coder/Crypto_Pulse.git
cd Crypto_Pulse
```

### 2. Install Dependencies

```bash
pip install -r requirements.txt
```

### 3. Run the Streamlit Application

```bash
streamlit run app.py
```
